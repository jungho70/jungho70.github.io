코드로 탐구하는 확률과 랜덤성

Intro Question Method Experiment Analysis Code

사이트 소개

이 사이트는 확률과 랜덤성을 코드로 실험하고 그 결과를 분석하는 탐구 공간입니다.

핵심 질문

랜덤은 정말 공정할까? 1000번 반복하면 결과는 50%로 수렴할까?

탐구 방법

파이썬을 이용해 동전 던지기 시뮬레이션을 반복 실행하고, 시행 횟수에 따른 확률 변화를 분석할 예정입니다.

실험 기록

동전 던지기 실험을 10번, 100번, 1000번으로 나누어 진행하고, 각 경우에서 앞면이 나오는 비율을 비교할 예정입니다.

분석

실험 결과를 통해 시행 횟수가 많아질수록 확률이 일정한 값으로 수렴하는지를 확인할 것입니다.

코드


import random

heads = 0
n = 1000

for i in range(n):
    if random.random() < 0.5:
        heads += 1

print("앞면 비율:", heads/n)
