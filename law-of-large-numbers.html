큰 수의 법칙과 확률 시뮬레이션

논문 분석 + 코드 실험 기록

1. 읽은 논문 / 자료

여기에 읽은 논문 또는 학술 자료의 제목, 저자, 출처를 적습니다.

2. 핵심 내용 정리

큰 수의 법칙은 시행 횟수가 많아질수록 실제 실험 결과의 평균이 이론적인 확률값에 가까워진다는 개념입니다.

3. 내가 이해한 점

확률은 짧은 시행에서는 불규칙하게 보일 수 있지만, 시행 횟수가 충분히 많아지면 일정한 패턴이 나타납니다. 따라서 확률은 개별 사건보다 반복 실험에서 더 잘 이해할 수 있습니다.

4. 추가 탐구 질문

동전을 10번, 100번, 1000번 던졌을 때 앞면의 비율은 정말 0.5에 가까워질까?

5. 파이썬 코드

import random

heads = 0
n = 1000

for i in range(n):
    if random.random() < 0.5:
        heads += 1

print("앞면 비율:", heads / n)

7. 분석 및 결론
