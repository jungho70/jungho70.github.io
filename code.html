코드 아카이브

탐구 과정에서 사용한 파이썬 코드 기록

1. 동전 던지기 기본 시뮬레이션

import random

heads = 0
n = 1000

for i in range(n):
    if random.random() < 0.5:
        heads += 1

print("앞면 횟수:", heads)
print("앞면 비율:", heads / n)

2. 시행 횟수별 비교 코드

import random

trials = [10, 100, 1000]

for n in trials:
    heads = 0

    for i in range(n):
        if random.random() < 0.5:
            heads += 1

    print(n, "회 시행:", heads / n)

3. 그래프 제작 예정 코드

import random
import matplotlib.pyplot as plt

heads = 0
ratios = []

for i in range(1, 1001):
    if random.random() < 0.5:
        heads += 1
    ratios.append(heads / i)

plt.plot(ratios)
plt.axhline(y=0.5, linestyle="--")
plt.xlabel("Number of trials")
plt.ylabel("Ratio of heads")
plt.title("Coin Toss Simulation")
plt.show()

4. 코드 설명

이 코드는 동전 던지기를 컴퓨터로 반복 실행하여 앞면이 나오는 비율을 계산한다. 시행 횟수가 증가할수록 앞면 비율이 0.5에 가까워지는지 확인할 수 있다.
